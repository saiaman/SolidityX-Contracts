# Quick tasks too help all you need

## Deploy a QRC20 ?

```shell
npx harhat deploy_ERC20 (--network YOUR_SELECTED_NETWORK ) YourTokenFullName YourTokenACRONYM YourTokenSupply
```

**for example**

```shell
npx harhat deploy_ERC20 Erc20_Test TERC20 1000000000
```
